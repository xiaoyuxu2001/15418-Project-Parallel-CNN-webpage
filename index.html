BACKGROUND

Training a Convolutional Neural Network (CNN) is a task that requires significant computational resources. Parallelizing this process has become essential to ensure that the training can be finished within a reasonable timeframe. In this project, we implemented an advanced parallelized CNN for image classification by integrating both data parallelism and model parallelism strategy. The parallelization is also planned to ensure optimal workload distribution and minimal communication overhead. We apply data parallelism on convolutional layers, and model parallelims on fully connected layers.
